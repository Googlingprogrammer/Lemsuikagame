SOUNDS FOLDER
=============

Place your sound files here and configure them in sound-config.js

BACKGROUND MUSIC:
- bgm.mp3      - Background music (loops infinitely)

SOUND EFFECTS:
- spawn.mp3    - Sound when fruit is dropped
- merge.mp3    - Sound when fruits merge
- victory.mp3  - Sound when player wins
- gameover.mp3 - Sound when game is over

SUPPORTED FORMATS: mp3, wav, ogg

NOTE: Sound effects have procedural placeholders if files are missing.
      BGM requires an actual file to play.

See sound-config.js for configuration options.
